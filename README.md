A mobile-first React Native app for quick idea tracking with threaded discussions, real-time collaboration, and a beautiful glassmorphic UI.

## Features

### Core
- **Quick Idea Capture** - Instantly save ideas with titles, descriptions, and icons
- **Threaded Replies** - Nested conversations with inline reply threads
- **Real-time Updates** - Live message delivery and read receipts via WebSocket
- **Emoji Reactions** - React to messages with emoji

### Collaboration
- **Team Invites** - Invite collaborators to your ideas
- **Permissions** - Control who can invite others or change backgrounds
- **Read Receipts** - See who has viewed your messages in real-time

### Media
- **File Attachments** - Share images, videos, and documents
- **Custom Stickers** - Upload and use personal sticker packs
- **Emoji Picker** - Full emoji keyboard with categories and search
- **Image/Video Backgrounds** - Personalize threads with custom backgrounds

### UI/UX
- **Glassmorphic Design** - Beautiful blur effects and glass cards
- **Dark/Light Mode** - System-aware theme switching
- **Animated Backgrounds** - Dynamic WebGL backgrounds on web
- **PWA Support** - Install as app on mobile devices
- **Push Notifications** - Stay updated on new messages

## Tech Stack

### Frontend
- React Native + Expo
- React Navigation
- Expo Blur, Image Picker, Video
- React Native Reanimated
- Socket.IO Client

### Backend
- Node.js + Express
- PostgreSQL
- Socket.IO (real-time)
- AWS S3 (file storage)
- JWT + Google OAuth 2.0
